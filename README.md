# eg-offline-jfx

Evergreen Offline UI in Java / JavaFX

## Running From Source (Ubuntu Version)

```sh
sudo apt install openjdk-11-jdk libopenjfx-java maven

git clone github.com:berick/eg-offline-jfx

cd eg-offline-jfx

mvn javafx:run -f pom.xml

```
